    Common failures
    
          
    
            
              
    
                Symptom
                Likely cause
              
    
                
            
              
    
                TickTick OAuth fails with a redirect mismatch
                The callback URI is not registered exactly, including http vs https.
              
    
              
    
                /debug/projects returns 401
                The DEBUG_TOKEN Bearer header is missing or wrong.
              
    
              
    
                GitHub webhook returns 401
                GITHUB_WEBHOOK_SECRET does not match the value configured in GitHub.
              
    
              
    
                Azure DevOps webhook returns 401
                The Service Hook basic auth username/password does not match your Worker secrets.
              
    
              
    
                Azure DevOps backfill says org/project must be configured
                You still have placeholder YOUR_ORG / YOUR_PROJECT values in wrangler.jsonc.
              
    
              
    
                Tasks go to the wrong TickTick list
                The source-specific list IDs are wrong, or you intentionally enabled shared-list mode but forgot.
              
    
              
    
                No tasks appear in production after local testing worked
                Production TickTick OAuth has not been completed yet. Local and remote D1 databases are separate.
              
    
              
    
                D1 errors mention missing tables
                Run npm run db:migrate:local or npm run db:migrate:remote.
